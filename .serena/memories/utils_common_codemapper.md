# Utility Toolkit
- `src/utils/dataManipulation.ts` offers hex/forge conversions and deep copy helpers.
- `src/utils/uint8Serialize.ts` handles Uint8Array <-> JSON serialization; `src/utils/pprint.ts` formats debug output; `getRemoteIP.ts` fetches external IPs.
- Aggregated through `src/utils/index.ts` for broad reuse.
